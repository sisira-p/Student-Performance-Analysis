# Student Performance Analysis | Python Project
## Overview
A data analysis project that examines factors influencing student exam performance using Python.
The project applies data cleaning, exploratory data analysis (EDA), and visualization techniques to derive actionable insights from academic data.

## Key Features
Cleaned and preprocessed raw student data
Performed Exploratory Data Analysis (EDA)
Classified students into performance levels (Low, Medium, High)
Created clear visualizations to support findings

## Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Academic performance dataset containing exam scores, study time, and attendance
**File:** https://www.kaggle.com/datasets/mosapabdelghany/student-performance-factors-dataset

## Methodology
Data preprocessing and validation
Statistical and visual analysis
Performance level classification using score ranges

## Results
Identified strong correlation between study time and exam scores
Attendance shown as a key contributor to performance
Performance categorization simplified insight interpretation

```
pip install pandas matplotlib seaborn
Run the Python script or open the notebook to view analysis and visual outputs.
```
